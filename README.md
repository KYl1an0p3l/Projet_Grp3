# Projet_Grp3

Pour démarrer le projet, il est nécessaire de lancer le serveur Node. Voici la procédure détaillée étape par étape.

---

### Étape 1 : Installation de Node.js

Node.js permet d'exécuter du JavaScript en dehors de votre navigateur (directement sur votre machine).

1. **Téléchargement :**
* Rendez-vous sur le site officiel : **[nodejs.org](https://nodejs.org)**.
* Téléchargez la version **LTS** (Long Term Support). C'est la version la plus stable et celle recommandée pour la plupart des utilisateurs.


2. **Installation :**
* **Windows/macOS :** Lancez l'installateur téléchargé et suivez les instructions à l'écran (cliquez sur "Suivant" jusqu'à la fin de l'installation).


### Étape 2 : Lancement du serveur

Une fois Node.js installé, ouvrez votre invite de commande (ou terminal) et suivez ces instructions :

1. **Accéder au dossier du projet :**
Il faut vous placer dans le répertoire où se trouve le projet à l'aide de la commande `cd`.
*Exemple (à adapter selon l'emplacement de votre dossier) :*

cd C:\Users\steph\Documents\GitHub\Projet_Grp3


2. **Démarrer le serveur :**
Une fois positionné dans le dossier via le terminal, tapez la commande suivante :


node server.js
